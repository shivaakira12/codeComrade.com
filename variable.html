VARIABLES AND VALUES

What is a Variable ?

Variable is a name given which is used to store the values.
In programming, a variable is a value that can change, depending on conditions or on information passed to the program.
Variables are the names you give to computer memory locations which are used to store values in a computer program.

Variable declaration in Javascript

var myName = "codeComrade"
var myAge = 23
console.log(myAge)
console.log(myAge)

Rules for Declaring the variable :

  1. The first letter could be a letter , underscore or dollar but it can’t be number
  2. The rest of the variable can be any number, letter or underscore but it can’t be any special characters including spaces.
  3. There is no limit for variable length.
  4. Variable names are case sensitive.
  5. We can’t use Javascript reserved words as variable names. For example : var , const etc
