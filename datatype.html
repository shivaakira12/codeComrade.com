DATA TYPES IN JAVASCRIPT

There are six data types in javascript that are primitives.

  1. undefined : typeof instance === "undefined"
  2. Boolean : typeof instance === "boolean"
  3. Number : typeof instance === "number"
  4. String: typeof instance === "string"
  5. Big int : typeof instance === "big int"
  6. Symbol : typeof instance === "symbol"

“Here the big int and symbol data types are used in ECMA script 2019-now.”

Examples of datatypes :

var myName = "codeComrade"
var myAge = 23
var itsRaining = true
var codeComrade
console.log(typeof(myName))
console.log(typeof(myAge))
console.log(typeof(itsRaining))
console.log(typeof(codeComrade))

Output

string
number
boolean
undefined
