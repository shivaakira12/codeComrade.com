INTRODUCTION TO JAVASCRIPT

What is Javascript ?

Javascript was initially created to make the web pages alive. The programs in this language called scripts.They can be written right in a web pages html and run automatically as web pages loads.Scripts are provided and executed as plain text.They don’t need special compilation to run. First name of javascript was “live script”.

Why Javascript ?

Javascript improves the user experience of the webpage by converting it from static web page into an interactive one. Javascript adds behaviour to web pages.

Applications of Javascript :

  1. Client side validation
  2. Manipulating HTML pages
  3. Server Application
  4. Backend Data Loading
  5. User notification
