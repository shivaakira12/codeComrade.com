CONDITIONALS IN JAVASCRIPT

Conditional statements in programming are used to check the condition if the condition is true then it will execute the statements.

These are the below conditional statements we use in Javascript.

If-statement :

If-statement checks the condition if the condition is true it executes the statements and gives the output.

var number = 5
if( number<10){
console.log("Yes number is less than 10)
}

Output

Yes number is less than 10

So here the number is 5 we declared and assigned and in if-block we are checking the condition if number<10 it is true right so it will executed the statement Yes the number is lesser than 10

else-statement :

Else statement executes the statement in else block when the if condition becomes false.

var name = "codeComrade"
if( name == "codeComrade"){
console.log(`Say hi to ${name}`)
}
else {
console.log(`Say No to ${name}`)
}

Output

Say hi to codeComrade

So here the output is Say bye to Codcomrade because the else block condition is not true i.e, myName variable is having “codComrade” and in if - block the condition is “codComrade==codeComrade” both are not same.
